#Projeto Java Web no Spring Boot
> Construir uma API REST simples com banco de dados usando Java e Spring Boot 

## Material de estudo DEVSUPERIOR
[DevSuperior - Escola de programação](https://devsuperior.com.br)

### Pré-requisitos

- Lógica de programação (qualquer linguagem)
- Programação orientada a objetos (qualquer linguagem)
- Ferramentas
  - Spring Tool Suite (STS)
  - Postman

### Objetivos do projeto

- Resgatar fundamentos de programação
- Colocar em prática esses fundamentos
- Criar um pequeno sistema com ferramentas e práticas de mercado
- Dar mais um passo em direção à preparação para o mercado

### Visão geral do sistema

 API REST de usuários e departamentos, com os seguintes casos de uso:

- Buscar todos usuários
- Buscar um usuário pelo seu id
- Inserir um novo usuário
